## 安装
```
pnpm install
```

### 热更新启动
```
pnpm start
pnpm dev
```

### 构建
```
pnpm build
```

### 规范检查(eslint + prettier)
```
pnpm lint
```

### 模板生成
```
pnpm generate
```
